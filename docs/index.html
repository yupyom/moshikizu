模式図・概念図を、簡便に。

資料に載せる図を素早く整えるためのドローイングツール。描いたものはぜんぶ後から編集でき、AI エージェントに作図を頼むこともできます。

macOS・Windows・ブラウザ版 / MIT License

汎用お絵かきアプリでは、ありません。

「PowerPoint の図形機能では物足りない、でも Illustrator は過剰」。Moshikizu は、その間のための道具です。少ない操作で整った図に特化しています。

01

図に特化した描画

矩形・角丸・楕円・折れ線（直交/曲線）・テキスト。破線、7種の矢印先端、図形を動かすと追従するコネクタ。

02

ぜんぶ後から編集できる

線のウェイポイント、ベジェ曲線のハンドル、画像の非破壊トリミング。「描き直し」がいりません。

03

表と数式、グラフ

=SUM/AVG が使える表と、表を参照する7種のグラフ。表を直せばグラフも追従します。

04

マルチキャンバスとマスター

16:9 / A4 などのプリセット、{page} が使えるマスターキャンバス、複数ページ PDF 書き出し。

05

AI エージェント連携（MCP）

Claude Code 等から作図・修正・プレビュー。ヘッドレス版とアプリ内ホスト版の2形態を同梱。

06

チーム共有（セルフホスト）

Node+SQLite の薄いサーバー。コメント・2要素認証・ゲスト共有リンク。Docker 一発で起動。

表もグラフも、頼める

2つの形態を同梱

開いている図をリアルタイム編集するアプリ内ホスト版と、ファイルを直接読み書きしてプレビュー画像を返すヘッドレス版。用途に合わせて選べます。

1行で、はじめる。

無料・オープンソース（MIT）。チーム用セルフホストサーバーも同梱しています。

# macOS / Linux

$ curl -fsSL https://raw.githubusercontent.com/yupyom/moshikizu/main/install.sh | bash

# Windows (PowerShell)

> irm https://raw.githubusercontent.com/yupyom/moshikizu/main/install.ps1 | iex

インストール不要で試すなら ブラウザ版プレイグラウンド へ（データはローカル保存）。
Node.js 20+ と git が必要です。コード署名なしのため、macOS は初回のみ 右クリック > 開く で起動してください。
手動セットアップやサーバー設置は マニュアル をご覧ください。
